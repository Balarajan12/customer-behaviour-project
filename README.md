# Customer Behaviour Analytics

This project analyzes customer purchasing behavior using Python, MySQL, and Power BI.

## Tech Stack
- Python (Pandas, NumPy)
- MySQL
- Power BI

## Project Flow
CSV Dataset → Python EDA → MySQL → Power BI Dashboard

## Files
- customer_data.csv – Raw dataset
- customer behaviour.sql – SQL analysis queries
- EDA.ipynb – Data cleaning & EDA in Python
- Customer_Dashboard.pbix – Power BI dashboard

## Insights
- Customer purchase patterns
- Subscription vs non-subscription revenue
- Category and age-based analysis
